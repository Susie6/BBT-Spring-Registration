## Structure
`./index.html` - 主页

`./signup.html` - 报名页

`./introduction.html` - 部门介绍页

`./check_login.html` - 内部人员登录页

`./check_show.html` - 内部人员查询页 

`./css/` - 存放样式表

`./js/` - 存放JavaScript

`./img/` - 存放图片

`./api/` - 存放后端程序

## Deployment
1. git clone到相应目录.
2. 编辑 `config-sample.php` ，记入数据库连接凭证，并将其更名为 `config.php`.
3. 执行 `install.php` 以测试数据库连接并建立表.
